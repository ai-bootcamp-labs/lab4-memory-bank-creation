# Agent Instructions for Personal Task Board

## Tech Stack
- Frontend: React 18 + Vite
- Backend: Node.js
- Language: TypeScript
- Storage: PostgreSQL

## Specification Structure
- PRDs are stored in: specs/prds/
- Epics are stored in: specs/epics/
- Stories are stored in: specs/stories/

## Naming Conventions
- PRDs: PRD-{feature-name}.md
- Epics: EPIC-{number}-{name}.md
- Stories: STORY-{epic}.{number}-{name}.md

## File Organization
Strictly follow the specification structure for outputs. Ensure all generated specs are specific and measurable.
